# REPIN'
Fitness Coaching Web App  
Based on multiple user inputs, this web app generates the most optimal workout plan and daily macro intake recommendation.

RUN THE WEB APP:
- download the files from GitHub to your machine
- open them with your IDE
- open 2 terminals in the IDE
- in the first one run the command: python app.py
- in the second one run the command: python -m http.server 8000
- After that the Web App should be working
- Search the URL: localhost:8000
- Enjoy!!
